## Quick Start (v2.1)

### Run with multi-agent support
```bash
export AGENTS=3
export MCP_ENABLED=true
export KNOWLEDGE_BACKEND=neo4j
python drop.py
```

### Access the full stack
- Web: `http://localhost:8000/`
- GraphQL: `http://localhost:8000/graphql`
- gRPC: `localhost:50051`
- Voice: WebSocket at `ws://localhost:8000/voice`

### Deploy as distributed Ocean
```bash
docker compose -f docker-compose.v2.yml up
# Brings up: API, Agents, Memory, MCP, Automation, Voice
```
